## Academic courses

### UC Berkeley - LLM Agents MOOC, Fall 2024

- **Provider:** UC Berkeley RDI / CS294-196
- **Level:** Intermediate to advanced
- **Format:** Full academic MOOC with lectures, slides, quizzes, and readings
- **Topics:** LLM reasoning, planning, tool use, RAG, AutoGen, DSPy, software agents, robotics, scientific discovery, privacy, safety, ethics, human-agent interaction, multi-agent collaboration
- **YouTube:** https://www.youtube.com/playlist?list=PLS01nW3RtgopsNLeM936V4TNSsvvVglLc
- **Course site:** http://rdi.berkeley.edu/llm-agents/f24

### UC Berkeley - Advanced LLM Agents MOOC, Spring 2025

- **Provider:** UC Berkeley RDI
- **Level:** Advanced
- **Format:** Full academic MOOC with lectures, slides, quizzes, and readings
- **Topics:** Advanced inference-time techniques, post-training, reasoning, memory, planning, mathematics agents, code agents, program verification, vulnerability detection, safe and secure agentic AI
- **YouTube:** https://www.youtube.com/playlist?list=PLS01nW3RtgorL3AW8REU9nGkzhvtn6Egn
- **Course site:** https://llmagents-learning.org/sp25

### UC Berkeley - Agentic AI, Fall 2025

- **Provider:** UC Berkeley RDI / CS294-196
- **Level:** Intermediate to advanced
- **Format:** Academic course / MOOC-style lecture series
- **Topics:** Agentic AI foundations, LLM agents overview, AI system design, post-training verifiable agents, agent evaluation, training agentic models, multi-agent AI, agents for scientific discovery, real-world deployment, safety and security
- **YouTube:** https://www.youtube.com/playlist?list=PLS01nW3RtgoqGkm4UeqNeZLccW-OGc1fJ
- **Course site:** https://rdi.berkeley.edu/agentic-ai/f25

### Stanford - CS329A Self-Improving AI Agents, Winter 2025

- **Provider:** Stanford
- **Level:** Advanced / research-oriented
- **Format:** Seminar-style lecture series
- **Topics:** Self-improving agents, coding agents, agent learning, evaluation, frontier research
- **YouTube:** https://www.youtube.com/playlist?list=PL3058ht9NqT1NG6Y663elpHSDh-AW1TIr

### Michigan State University - CSE 491/895 AI Agents

- **Provider:** Michigan State University / Prof. Mohammad Ghassemi
- **Level:** Beginner to intermediate
- **Format:** University lecture series
- **Topics:** Prompt engineering, agentic design patterns, prompt chaining, routing, reflection, memory, learning, adaptation, Model Context Protocol, evaluation, cost, scalability
- **Channel:** https://www.youtube.com/@ghassemi/videos
- **Starting lecture:** https://www.youtube.com/watch?v=Xn-OyfNQVv4

### CMU - Advanced NLP agent lectures

- **Provider:** Carnegie Mellon University / Graham Neubig and collaborators
- **Level:** Intermediate to advanced
- **Format:** Advanced NLP course with dedicated lectures on tools and agents
- **Topics:** Tool use, LLM agent basics, agent architectures, agent environments, multi-agent communication
- **Course playlist:** https://www.youtube.com/playlist?list=PL8PYTP1V4I8D4BeyjwWczukWq9d8PNyZp

- **Format:** 10-lesson course with videos and code samples

## Suggested learning paths

### For beginners who want to build quickly

1. Microsoft - AI Agents for Beginners
2. Hugging Face - AI Agents Course
3. freeCodeCamp - LangGraph Complete Course
4. Google / Kaggle - 5-Day AI Agents Intensive

### For academic depth

1. UC Berkeley - LLM Agents MOOC, Fall 2024
2. UC Berkeley - Advanced LLM Agents MOOC, Spring 2025
3. UC Berkeley - Agentic AI, Fall 2025
4. Stanford - CS329A Self-Improving AI Agents

### For engineers building production agents

1. Microsoft - AI Agents for Beginners
2. freeCodeCamp - LangGraph Complete Course
3. Building Multi-Agent Systems with AutoGen, LangGraph & CrewAI
4. UC Berkeley lectures on evaluation, safety, production deployment, and multi-agent AI

## Contributing

Contributions are welcome. Please open a pull request with:

- Course title
- Provider / institution
- YouTube link
- Course site, if available
- Level: beginner, intermediate, advanced, or research
- Short topic summary
- Whether it is a full course, playlist, MOOC, or single lecture

Please keep entries focused on free, substantial courses or lecture series. Short promo videos should go under single-lecture supplements only if they are high-signal.
