# Contributing

Thanks for helping improve this list.

## What belongs here

Please add only resources that are:

- Free to watch or access
- Substantial enough to be a course, MOOC, playlist, or serious lecture series
- Focused on AI agents, LLM agents, agentic AI, multi-agent systems, tool use, planning, evaluation, or production deployment

## Entry format

Use this format:

```md
### Course title

- **Provider:** Institution or creator
- **Level:** Beginner / intermediate / advanced / research
- **Format:** Course / playlist / MOOC / lecture series
- **Topics:** Short list of topics
- **YouTube:** URL
- **Course site:** URL, if available
```

## Quality bar

Prefer:

- University courses
- Courses with source code, assignments, readings, or slides
- Full playlists over isolated videos
- Direct YouTube and course-site links

Avoid:

- Paid-only courses
- Thin marketing videos
- Duplicates
- Broken links
